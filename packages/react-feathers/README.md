# @fmbm/react-feathers

> React bindings for [Feathers](https://feathersjs.com/)

## Install

```bash
yarn add @fmbm/react-feathers
```

## Example

Clone this repo

```bash
yarn && yarn workspace @fmbm/react-feathers build && yarn workspace example start
```
